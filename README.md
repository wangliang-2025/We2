# 💕 我们的小屋 · Our Little Home

> 一个属于你们两个人的秘密花园 · A secret garden just for two

一个用 **Next.js 14** 构建的情侣恋爱网站，主题风格是 **马卡龙撞色 + 液态玻璃质感**，支持 **中英双语** 切换，**响应式设计**（手机/平板/电脑都好看），**完全免费部署**到 Vercel。

### Vercel 一键部署

### 方式 A：用 GitHub 一键部署（推荐）

#### 第 1 步：注册 GitHub
1. 打开 https://github.com 注册一个账号（如果已有可以跳过）

#### 第 2 步：把代码上传到 GitHub
**最简单的办法**（不用装 Git）：
1. 在 GitHub 右上角点 `+` → `New repository`
2. 仓库名随便填，比如 `our-little-home`，**勾选** "Private"（私密），点 `Create repository`
3. 在新仓库页面点 `uploading an existing file`
4. 把这个项目里**除了 `node_modules` 和 `.next` 之外**的所有文件夹拖进去
5. 拉到底点 `Commit changes`

#### 第 3 步：注册 Vercel 并部署
1. 打开 https://vercel.com → 用 GitHub 账号登录（最方便）
2. 进首页点 `Add New...` → `Project`
3. 找到刚才的 `our-little-home` 仓库，点 `Import`
4. **不用改任何配置**，直接点 `Deploy`
5. 等大概 1-2 分钟，看到 🎉 就成功了
6. 复制生成的网址（类似 `our-little-home-xxx.vercel.app`），就是你们的小屋了！

#### 第 4 步（可选）：自定义域名
- 如果想要自定义二级域名（免费），在 Vercel 的项目设置 → `Domains` 里把默认域名改成你喜欢的，比如 `wo-ai-ni.vercel.app`

## 🎨 首次使用配置

部署成功后，在网站里打开 **设置页**，做以下配置：

1. **专属称呼**：填上你和 TA 的昵称
2. **在一起的日期**：选择你们正式在一起的那天
3. **城市**：填上各自所在的城市（双城天气会显示）
4. **暗号**：设置一个只有你们才知道的暗号（在"游戏"页面输入可以触发爱心烟花特效）
5. **主题/语言**：选择喜欢的风格

填完后回到首页，时间计数器就开始为你们计时啦 ⏱️💕

## ⚠️ 关于数据存储

**当前版本所有数据存在浏览器本地（localStorage）**，这意味着：

✅ 优点：
- **完全免费**，零服务器成本
- 数据在你自己的浏览器，**绝对私密**
- 部署到 Vercel 也不用设置任何数据库

⚠️ 限制：
- 数据只在**当前浏览器/设备**可见，换设备/换浏览器看不到
- 清浏览器缓存 = 数据丢失（**所以一定要定期导出备份！**）
- 你和 TA 的数据**不会自动同步**（除非你们用同一台设备）

### 备份小技巧
打开 **设置页** → 滚到底 → 点 **"导出全部数据"**，会下载一个 `.json` 文件，**建议每周导出一次**存到云盘里。

### 想要双方实时同步怎么办？
当前版本是为了"零成本快速跑起来"做的纯前端版本。如果你们以后想要真正的双向同步（异地恋实时聊天等），有两个升级路径：

1. **接入 Supabase**（推荐，免费额度够 2 个人用）：
   - 在 Supabase 创建项目，把 `src/lib/storage.ts` 里的本地存储调用换成 Supabase API
   - 添加用户登录（绑定两个账号）
2. **接入 Firebase**（Google 的方案，类似）

代码已经做了抽象设计（`store.xxx.list/add/remove` 这种统一接口），后续切换到云端只需要改一个文件。

## 💡 常见问题

**Q：为什么首页时间不动？**
A：去 设置页 → 设置"在一起的日期" → 保存，刷新一下即可。

**Q：照片上传后清缓存就没了？**
A：是的，这是浏览器 localStorage 的特性。**强烈建议定期在设置页导出数据备份**。

**Q：异地恋的 TA 看不到我新加的内容？**
A：当前版本只有本地存储。两人想看到同样数据的话，要么同一台设备/浏览器，要么自己升级到 Supabase 后端（参考上面"想要双方实时同步"章节）。

**Q：可以加更多功能吗？**
A：当然！代码注释清晰，所有功能模块都是独立的，你可以让 AI 帮你继续加。

**Q：能改主题颜色吗？**
A：可以。打开 `tailwind.config.ts`，找到 `macaron` 那一段，改颜色值即可。
